🐾 VetVida — Clínica Veterinaria & Tienda Virtual

Sitio web profesional para clínica veterinaria con sistema de turnos online y tienda de productos para mascotas.

📋 Descripción
VetVida es un sitio web completo para una clínica veterinaria ficticia ubicada en Rosario Santa fe, Argentina. El proyecto está compuesto por dos páginas principales: una landing page institucional con formulario de reserva de turnos, y una tienda virtual con catálogo de productos, carrito de compras y sistema de pago simulado.

🗂️ Estructura del proyecto
vetvida/
├── index.html        # Landing page principal (clínica)
├── tienda.html       # Tienda virtual de productos
├── styles.css        # Estilos de index.html
└── script.js         # JavaScript de index.html

✅ Cursor personalizado con efecto lag (inercia)
✅ Navegación fija con efecto de scroll
✅ Sección Hero con contadores animados (scroll-triggered)
✅ Grilla de servicios con hover effects
✅ Cards de equipo médico con overlay interactivo
✅ Sección de testimonios
✅ Formulario de reserva de turno con validación
✅ Efecto parallax en texto de fondo
✅ Scroll Reveal con IntersectionObserver
✅ Toast de notificación animado
✅ Diseño responsive (mobile-first)

Tienda Virtual (tienda.html)

✅ Pantalla de carga animada
✅ Slider/carrusel automático con 4 banners (touch/swipe support)
